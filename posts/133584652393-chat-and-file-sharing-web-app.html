Chat and File Sharing Web App

Cette petite application web était simplement une occasion d'essayer NodeJS et les nouvelles fonctionnalités founies en HTML5 : l'envoi de fichier sans rechargement, le drag-and-drop, et les WebSockets (avec Socket.IO). Le résultat est un chat en ligne assez complet, et un “répertoire” public de fichiers, le tout sur une seule page.

Schedule (IUT of Blois)

The official one was outdated and unsuitable for phones, I created an alternative schedule, optimized for mobile and that can operate offline. It updates itself automatically when the browser is online, by retriving its information on the main site.

The source code is still available on GitHub.

Having had to do some reverse engineering to automatically update schedules, and this may not appeal to the university, I didn’t really spoke about this application to anyone apart from some of my classmates.

The Client

Offline functioning is assured by the use of HTML5 Application Cache, and the layout is responsive thanks to CSS3. The requests to the server are asynchronous (AJAX with ZeptoJS).

The Server

It is written in JavaScript with NodeJS and Express. The database of schedules is managed with MongoDB.

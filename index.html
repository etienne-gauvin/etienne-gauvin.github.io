Projets personnels

Ceux-ci se trouvent pour la plupart sur mon compte GitHub :

  • API Allociné Helper

    L'API Allociné Helper est une bibliothèque PHP permettant d'utiliser simplement l'API privée d'AlloCiné. Celle-ci n'étant normalement accessible qu'aux applications mobiles d'AlloCiné, un peu de reverse-engineering a été nécessaire pour comprendre son fonctionnement. Cette bibliothèque peut être installée avec Composer.

    Page du projet sur GitHub
    Page du projet sur Packagist

  • Indapocket

    Ce petit logiciel créé en JavaScript, avec Electron et youtube-dl, permet de télécharger rapidement et simplement de la vidéo et de l'audio depuis la plupart des sites les plus connus (Youtube, Vimeo, SoundCloud…).

    • JavaScript (ES6)
    • Node.js
    • Electron

    Page du projet sur GitHub

  • Clicliclick

    • Jeu
    • JavaScript
    • API Canvas
    • NodeJS
    • Socket.IO

    Un petit jeu de plateau multijoueur en ligne, réalisé en JavaScript et à l'aide de l'API Canvas pour l'affichage, de NodeJS pour la partie serveur, et de Socket.IO pour la communicationen temps réel. Le but du jeu est de capturer plus de points que son adversaire avant la fin du temps imparti ; il est semblable au Rollit, à ceci près que tous les joueurs peuvent modifier le plateau en même temps. Le jeu se base donc sur la rapidité d'action et de réflexion.

    • Code source sur GitHub

  • Partage de fichiers

    • HTML5/CSS3
    • JavaScript
    • NodeJS
    • Socket.IO

    Cette petite application web était simplement une occasion d'essayer NodeJS et les nouvelles fonctionnalités founies en HTML5 : l'envoi de fichier sans rechargement, le drag-and-drop, et les WebSockets (avec Socket.IO). Le résultat est un chat en ligne assez complet, et un “répertoire” public de fichiers, le tout sur une seule page.
